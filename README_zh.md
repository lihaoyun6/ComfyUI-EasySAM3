# ComfyUI-EasySAM3  
在 ComfyUI 中使用 SAM3 的最简方案 (支持视频/图片)  
**[[📃English](./README.md)]**

## 安装  

#### 安装节点:  
```bash
cd ComfyUI/custom_nodes
git clone https://github.com/lihaoyun6/ComfyUI-EasySAM3.git
python -m pip install ComfyUI-EasySAM3/requirements.txt
```
